## ğŸ’» How to Run

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required

### Installation Guide

1. **Clone or download the project**
   ```bash
   git clone [repository-url]
   cd psd2web_dev_vibe
   ```

   - Open browser and go to: `http://localhost:8000`
